UX/UI Design

Neuron App

Singapore has great infrastructure so a bike-sharing system is very popular in this city. This case study is about how I approached to create a booking and tracking experience easier that need it for the rider as never.

  • Design Research
  • Wireframing
  • Prototyping
  • UX/UI Design
  • UI Animation

The Challenge

Neuron brand already has several services by one brand, it has focused on another one. We were tasked with design the modern and delightful iOS app for mobility sharing system so that the visual style will be consistent with the general brand and will open new opportunities for Neuron’s system users.

The Problem

The goal was to create a functional and complex app with delightful experience and allow users to be happy to share rides with friends and have rides for points.

Competitor Audit

  • The competitor doesn't provide scanning QR code
  • Competitors don’t provide scooters for the users
  • Competitors don't have the "Book" device feature
  • Competitors don’t have a gamification approach

User Flow

I created a chart for User Flow based on the main features and the user’s needs. To make booking the device easier and focus the user on finding a good place to have a ride from, the user has an access to do this upon opening the app. The dashboard feature is one of the important feature created to track the riding, that can be stopped whenever the user wants.

Solutions

  • The interactive map is the first upon opening the app. This allows the user quickly book a device.
  • The "book" feature implemented so that the user can Cancel their booking after confirmation.
  • The interactive map provides the user information about each docking point and free devices for booking. It allows the user easily book device in any place of the city.
  • The dashboard UI is created so that the user can easily track the data about the ride and be confident on any device.
